== Description ==
In this plugin user can submit new bug with some details and user can maintain from WordPress admin panel. also user can put Bug Tracker submit form from front-end using shotcode .
`Features`
<ul>
<li>Add multiple bug os.</li>
<li>Add multiple brwoser.</li>
<li>Add Bug.</li>
<li>No Need Of Knowledge of PHP, HTML or CSS. But for those having knowledge can easly modify or customize.</li>
</ul>

== Installation ==

1. Upload the mitm-bugtracker folder to the /wp-content/plugins/ directory `OR` Upload from Plugins > Add New > Upload zip file.
2. Go to the 'Plugins' page of your WordPress administration area and activate the plugin.
3. Use the shortcode `[mitmbugtracker]` in the content area of a page or post where you want the bug tracker to appear.
